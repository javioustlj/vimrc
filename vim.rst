搜索

1. 查找光标所在字符串
光标停留在想要查找的字符串上面，然后输入Shift + \*, 就可以快速查找选中的单词

2. 查找寄存器中的字符串
/ "0 或者/ Ctrl + R 0


vim mode
========

1. Normal
2. Edit::

    i/I 是insert，在光标所在字符前/行首插入
    o/O 是open，在光标所在行的下面/上面新起一行
    a/A 是append，在光标所在字符后/行尾插入
    s/S 是substitute，删除当前字符并开始插入，删除光标所在行并开始插入
    r/R 是replace，替换光标所在字符/至行尾所有字符

3. Command::

    : 执行命令
    / 向文件结尾搜索
    ? 向文件开头搜索

4. Visual::

    v 连续字符
    V 行
    Ctrl + V 列编辑
